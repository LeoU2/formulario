2) Responda as perguntas abaixo:

A) Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

O atributo "placeholder" especifica uma dica curta que descreve o valor esperado de um campo de entrada e ou área de texto.

B) Na tag <input> com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Este tributo booleano, especifica que um elemento <input> deve ser pré-selecionado (checked) quando a página for carregada.

C) Na tag <input> para que serve o atributo "value"?

O atributo "value" especifica um valor inicial para um campo de entrada.

D) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

O atributo "radio" são geralmente usados por padrão como círculos renderizados onde apenas um pode ser preenchido ou destacado quando selecionado.

E) Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

A tag "label" é usado para criar um rótulo, etiqueta ou marcação, já o atributo "for" é o único "id" para um elemento rotulável relacionado ao formulário no mesmo documento que o <label>, qualquer "label" elemento dado pode ser associado a apenas um controle de formulário.

F) Na tag <option> para que serve o atributo "selected"?

O atributo "selected" serve para escolher uma seleção na tag "option", para que este seja o valor padrão selecionado no "seleted" elemento quando a página for carregada.

G) Nas tags de formulários, para que serve o atributo "required"?

O atributo booleano "required" é um campo específico numa área de texto obrigatóriamente preenchida para enviar um formulário.

H) Na tag <form> para que serve o atributo "action"?

O atributo "action" especifica para onde enviar os dados do formulário qunado é enviado.

I) Na tag <form> para que serve o atributo "method"?

O atributo "method" especifica como enviar dados de formulário para a página especificada no atributo "action".

J) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

O valor na tag "form" deve ser usado o "hidden" que é booleano e indica se um elemento é ou não relevante e pode ser usado para esconder elementos de página que não podem ser usados até que o processo de login seja completo.

K) Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

O atributo "post", não tem limite de tamanho. Exe.: <form method="post", <input type="text">, <input type="submit"> </form>.
